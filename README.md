## Pour démarrer le programme:
Sous VSCode:<br>
- Cliquez sur le bouton **Run** qui apparaît dans le fichier **main.rs**.<br>

Pour lancer le programme à la main depuis un terminal:<br>
```shell
  rustc main.rs -o mon_programme
  ./mon_programme
```
